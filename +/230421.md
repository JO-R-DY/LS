# 433 - [내륙] [내륙_광주본부]송도 인더쥬 실내동물원 컨텐츠 수정
- 작업시간 기준 : 30분미만으로 걸리는 업무
- 수정해야할 파일 : 1-1인더쥬동물원(요금표, 가이드 수정)
- 경로
```
\\192.168.50.222\mnt\DESIGN\2 내륙\송도 인더쥬 실내동물원
```
※ 1-1인트로, 2-1요금, 3-1가이드, 4-1리뷰 분리하기전 작업물이라 1-1통으로 작업되어있음.
